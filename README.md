# 🤖 IntellMeet

AI-Powered Enterprise Meeting & Collaboration Platform built with the MERN Stack.

## 📌 Overview

IntellMeet is a full-stack enterprise meeting platform that enables users to create and join video meetings, collaborate with teams, communicate in real time, and leverage AI-powered meeting intelligence.

## 🚀 Features

- 🔐 User Authentication (JWT)
- 👥 User Registration & Login
- 📹 Real-Time Video Meetings
- 💬 Real-Time Team Chat
- 🤝 Team Collaboration
- 📝 AI Meeting Summaries
- ✅ Smart Action Items
- 📅 Meeting Scheduling
- 🔔 Notifications
- 📂 Meeting History
- 📱 Responsive UI
-

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kalevaishnavi04/IntellMeet.git
```

### Install Dependencies

Client

```bash
cd client
npm install
```

Server

```bash
cd ../server
npm install


## ▶️ Run Project

Backend

```bash
cd server
npm run dev
```

Frontend

```bash
cd client
npm run dev


## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```
PORT=5000

MONGODB_URI=Your_MongoDB_URI

JWT_SECRET=Your_JWT_Secret

OPENAI_API_KEY=Your_OpenAI_Key
```
